Quick Installation

Get started in minutes with our simple setup process

1

Clone Repository

git clone https://github.com/yourusername/infra-chat.git
cd infra-chat
2

Install Backend Dependencies

cd backend
pip install -r requirements.txt
python ingest_minimal.py
3

Install Frontend Dependencies

cd ../frontend
npm install
4

Start Application

# Terminal 1 - Backend
python app_minimal.py

# Terminal 2 - Frontend
npm run dev
